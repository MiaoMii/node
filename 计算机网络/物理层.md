## 信道

> 某一个方向传送信息的媒体

|          |                                |
| -------- | ------------------------------ |
| 单工通信 | 只能有一个方向的通信           |
| 双工     | 两个方向通信，但不能同时发和接 |
| 全双工   | 双方可以同时发送和接收         |
